# 🇮🇩 Indonesian IoT Journals Directory

A curated directory of **10 open-access Indonesian journals** covering Internet of Things (IoT) research, published 2020 and beyond.

## 🌐 Live Site

> [Deploy to Vercel](https://vercel.com/) — *URL will be available after deployment*

## 📋 About

This directory helps researchers, students, and practitioners find freely accessible Indonesian journals that publish IoT research. All listed journals are:

- ✅ **Open Access** — free to read
- ✅ **Active since 2020+** — recent and ongoing publications
- ✅ **Indexed** — in SINTA (Indonesia's national ranking) and/or Scopus
- ✅ **IoT Coverage** — publish Internet of Things research

## 🏗️ Tech Stack

- Pure HTML / CSS / JS — no frameworks, no build step
- Deployed on **Vercel** (static site)
- Responsive design, dark theme
- Filter by accreditation level + full-text search

## 🚀 Deploy

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
cd IoT-Journals
vercel --prod
```
